## 1 基础知识

### 1.1 项目结构

#### 拓扑结构（描述连接关系与边界，不关心具体形状）

- **vertex**：顶点，零维拓扑元素，表示边的端点或面的角点。
- **edge**：边，一维拓扑元素，有方向，以两个 vertex 为边界，对应几何上的 curve。
- **face**：面，二维拓扑元素，由封闭的 edge 环围成，对应几何上的 surface。
- **shell**：壳，由一组 face 通过边邻接构成，形成封闭或开放的“面集合”。
- **lump**：块，三维拓扑元素，由一组封闭 shell 围成，表示一个连通的实体区域。

#### 几何结构（描述具体形状与位置，不包含连接关系）

- **point**：点，零维几何元素，由坐标定义。
- **curve**：曲线，一维几何元素，如直线、圆弧、样条线。
- **surface**：曲面，二维几何元素，如平面、圆柱面、NURBS曲面。

### 拓扑与几何的关联（拓扑承载几何）

| 拓扑元素 | 对应的几何元素 | 说明                                                |
| :------- | :------------- | :-------------------------------------------------- |
| vertex   | point          | vertex 可在某个 point 上                            |
| edge     | curve          | edge 沿着某条 curve 的一段                          |
| face     | surface        | face 位于某块 surface 上                            |
| shell    | （无直接对应） | 多个 face 共享几何 surface，但 shell 只记录拓扑关系 |
| lump     | （无直接对应） | 表示封闭空间，无直接几何映射                        |

### 1.2 偏置操作

偏置主要包含Offset、Hollow、Thicken。

​	**Offset（偏置）** ：将几何元素（点、线、面、体）沿法向方向移动指定距离，生成新几何。

​	**Hollow（抽壳）**：将实体内部挖空，保留指定厚度的薄壳。

​	**Thicken（加厚）**：给曲面或薄片实体增加厚度，生成带体积的实体。

其中，Offset 是基础操作；Hollow 可理解为对实体内表面进行“负向偏置并移除材料”；Thicken 可理解为对曲面向两侧或单侧偏置后封闭成实体。

**三者的对比与关系**

| 功能    | 输入       | 输出            | 是否改变拓扑   | 典型问题           |
| :------ | :--------- | :-------------- | :------------- | :----------------- |
| offset  | 面/边/实体 | 偏置面/扩大实体 | 通常不变       | 自交、法向突变     |
| hollow  | 封闭实体   | 薄壳实体        | 是（移除材料） | 小特征、厚度不均   |
| thicken | 曲面/片体  | 实体            | 是（增加体）   | 曲率过大、端面闭合 |

**核心共性**：都依赖**法向偏置**作为底层几何计算，区别在于拓扑操作（挖空、加侧壁、封闭端面）及应用对象（实体整体 vs 曲面）。

### 1.2.1 Offset（偏置）

Offset 是 Hollow 和 Thicken 的基础，其通用流程包含三个关键子操作：

​	1.Trim（修剪）：去除偏置后曲面产生的自交、多余部分，保持边界连续。

​	2.Repair（修复）：修复自交、缝隙、碎面等缺陷，重建拓扑一致性。

​	3.Extend（延展）：偏置后曲面可能缩短，无法与相邻面接合，需让偏置面边界延伸到可正确连接的状态。

基本流程：选定面 → 沿法向偏移 → 自动/手动 Trim → 检查并 Repair 错误 → 对不足区域 Extend → 缝合为新实体/曲面。

### 1.2.2 Hollow（抽壳）

将实体变成薄壳结构，保留一个或多个开口面。Hollow 本质是“对实体所有面同时做 offset，删除内部材料”。可分为**向内偏移**、**向外偏移**。

​	1.向内偏移：先选择要移除的面（开口面），后将剩余实体表面沿法向向内偏置壁厚距离，最后生成内部空腔、保留开口区域。

​	2.向外偏移：先选择保持封闭的原始实体，然后将实体表面沿法向向外偏置壁厚距离，最后将原实体定义为空腔内侧壁，新偏置面定义为外侧壁。

### 1.2.3 Thicken（加厚）

Thicken 从曲面开始新增体，Hollow 从实体开始掏空。

​	1.**单侧加厚**：沿曲面法向仅向一侧增加厚度。

​	2.**双侧加厚**：沿曲面法向两侧同时增加厚度，生成对称实体。

​	3.**沿方向加厚**：不沿法向，而是沿指定矢量方向拉伸厚度。
